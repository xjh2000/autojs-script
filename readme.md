# Aoutjs
不需要Root权限的JavaScript自动化软件
## 安装
```bash
npm i -g autojs-dev
```

支持的指令：
```bash
autojs create <filename> # 根据模板创建脚本文件

autojs cap [filename] # 对adb列表的第一个设备进行截图

autojs new [projectName] # 新建Auto.js项目，可选用npm或yarn进行npm install操作

autojs build # 根据配置文件编译脚本

autojs img # 启动找图素材加载服务
```
